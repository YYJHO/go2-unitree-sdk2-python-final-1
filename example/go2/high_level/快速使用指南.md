## 🎮 操作技巧

### 基础操作

```
1. 让机器人站立：
   基础控制 → 点击"站立"
   
2. 移动机器人：
   运动控制 → 长按"↑前进"
   
3. 停止移动：
   松开按钮（自动停止）
   
4. 让机器人休息：
   基础控制 → 点击"坐下"
```

### 姿态控制

```
1. 身体向右倾斜：
   基础控制 → Euler → Roll输入"0.2"
   → 点击"设置姿态"
   
2. 恢复水平：
   基础控制 → 点击"重置姿态"
```

### 速度调节

```
基础控制 → 速度等级
- 选择"慢速" → 安全学习
- 选择"中速" → 日常使用
- 选择"快速" → 运动模式
```

### 互动表演

```
高级技能 → 互动动作
- 点击"招手" → 打招呼
- 点击"比心" → 可爱动作
- 点击"舞蹈1" → 跳舞
```

## 🔧 常见问题

### Q: 程序启动后卡死？
A: 已修复！使用多线程架构，不会卡死。

### Q: 如何持续移动？
A: 长按方向键，松开自动停止。

### Q: 没有机器人能测试吗？
A: 可以！使用 `--mock` 参数启动调试模式。

### Q: 如何切换语言？
A: 顶部菜单 → 语言 → 选择中文或English。

### Q: 状态显示"未准备好"？
A: 点击"准备运动"按钮，或查看"机器人状态"标签页。

### Q: PySide2和PyQt5有什么区别？
A: PySide2是免费的Qt官方绑定，PyQt5商业使用需付费。
